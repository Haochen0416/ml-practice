# ML Practice

Machine learning algorithms and data analysis projects implemented in Python.

## Projects

### 1. Linear Regression (`hello_ml.py`)
Implements linear regression from scratch without any ML libraries.
- Calculates mean, variance, and covariance manually
- Predicts exam scores based on hours studied

### 2. K-means Clustering (`kmeans.py`)
Implements K-means clustering from scratch.
- Euclidean distance calculation
- Centroid initialization and update
- Convergence detection

### 3. Pandas Data Analysis (`pandas_practice.py`)
Data cleaning and analysis using Pandas.
- DataFrame creation and inspection
- Missing value detection and filling
- Conditional filtering, sorting, and groupby

### 4. Scikit-learn ML Pipeline (`sklearn_pipeline.py`)
Complete ML pipeline using scikit-learn.
- Train/test split and feature scaling
- Logistic Regression model training and evaluation
- Prediction on new data with confidence scores

### 5. Data Visualization (`visualization.py`)
Four-panel analysis dashboard using Matplotlib.
- Bar chart, histogram, scatter plot, box plot
- Visual comparison of pass/fail student performance

### 6. Titanic Survival Prediction (`titanic_classification.py`)
End-to-end classification project with feature engineering.
- Data cleaning and missing value handling
- Feature engineering (family size, is_alone)
- Model comparison: Logistic Regression vs Random Forest
- Feature importance visualization

## Tech Stack
Python | Pandas | NumPy | scikit-learn | Matplotlib
